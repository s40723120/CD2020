Week12

Virtualbox設定網路

新增虛擬主機後點settings

改成下面格式

為了抓出DHCP的IP所以下面指令

ifconfig

上面顯示沒有安裝net-tools，所以打

sudo apt install net-tools

出現網路錯誤 之後查詢網路設定

先到/etc/netplan目錄下

cd /etc/netplan

然後檢查 00-installer-config.yaml 檔案內的設定

sudo vi 00-installer-config.yaml

00-installer-config.yaml內容

# This is the network config written by 'subiquity'
network:
    ethernets:
        enpes3:
            dhcp4: true
            dhcp6: true
            nameservers:
                addresses:
                    - 2001:b000:168::1
    version: 2

遍輯完後按ESC打:wq退出，之後更新netplan設定
sudo netplan apply

之後到/etc/apt/apt.conf.d 目錄下檢查有沒有proxy.conf檔案(通常沒有)

cd /etc/apt/apt.conf.d

新增proxy.conf

sudo vi proxy.conf

proxy.conf的檔案內容

Acquire::http::proxy "http://[2001:288:6004:17::填port號]:3128";

到Porxy查詢port號

打完後更新apt
sudo apt update



之後就可以下

sudo apt install net-tools

ifconfig

上面找出 fe80:a00:27ff:fef6:9b8a 是我的Guest IP
